# Asynch Programming
A quick and simple demo to illustrate asynchronous programming concepts

## Local setup

Just run an HTTP server, for example:

```
python -m SimpleHTTPServer 2222
```

The server local server will be available at:

http://localhost:2222

## Demos

### Promises
